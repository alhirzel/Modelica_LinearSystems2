Modelica_LinearSystems2.Controller.Templates

Package of templates used for demonstration purposes

Information

This package contains template models designed to build up test models for controllers. The models have to be extended with particular models to create a virtual test rig of a controller.

Extends from Modelica.Icons.BasesPackage (Icon for packages containing base classes).
